# Shopify Orders Create Client for Server GTM

Custom Server-side Google Tag Manager (sGTM) Client Template for Shopify `orders/create` webhooks.

## Features

- Purchase Event Creation
- Dynamic Transaction ID
- Dynamic Purchase Value
- Dynamic Currency
- Dynamic Product Items
- Dynamic FBP
- Dynamic FBC
- Dynamic User Agent
- Dynamic IP Address
- Meta Conversions API Ready

## Webhook Topic

orders/create

## Supported Platforms

- Shopify
- Server GTM
- Meta Conversions API

## Installation

1. Create a Custom Client Template in Server GTM
2. Add the provided code
3. Create a path (example: `/orders-create`)
4. Publish the container
5. Configure Shopify Webhook to point to:

https://your-sgtm-domain.com/orders-create
